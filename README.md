# Note Manager

A simple, lightweight, and efficient web application designed to help you manage your daily notes with ease. Built from scratch using vanilla web technologies, it focuses on speed and simplicity while keeping your data safe locally.

## ✨ Features

- **Add Notes:** Quickly write and save new notes through a clean input interface.
- **Local Storage:** Your notes are automatically saved in your browser's local storage. They will persist even after you close or refresh the page.
- **Responsive UI:** A clean and user-friendly interface that works across different screen sizes.

## 🛠️ Tech Stack

- **HTML** - Structure
- **CSS** - Styling and Layout
- **JavaScript (Vanilla)** - Logic, DOM manipulation, and LocalStorage management

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps:

1.  Clone the repository or download the source code.
2.  Open the `index.html` file in your favorite modern web browser.
3.  Start adding your notes!

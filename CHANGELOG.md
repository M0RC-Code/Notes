## [v0.1.0] - 2026-07-15

### Added

- **Core Note Logic:** Implemented the main functionality to add new notes via the input field and submit button.
- **Initial UI Styling:** Set up the basic layout and styling structure for the note manager interface.
- **LocalStorage Persistence:** Integrated browser LocalStorage to save notes automatically, ensuring data is not lost on page refresh.
